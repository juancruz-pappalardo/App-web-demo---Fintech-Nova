FinTech Nova — Demo web
======================

Archivos incluidos:
 - index.html  : Página principal de demostración.
 - styles.css  : Estilos modernos, responsive.
 - app.js      : Interacciones JS (demo estático).
 - README.md   : Este archivo.

Descripción:
Esta es una demo estática pensada para presentaciones y finales de materia. Incluye:
 - Hero con dato de impacto
 - Mock de app con acciones (enviar, pagar QR, solicitar microcrédito)
 - Secciones: Producto, Impacto, Tecnología, Equipo y Contacto
 - Modal con calculadora de cuota (modelo simplificado, demo)
